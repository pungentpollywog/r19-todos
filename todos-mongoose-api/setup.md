# Setup

## Dependencies

This upgrades express, mongoose, body-parser and installs the other dependencies needed (bcrypt, jsonwebtoken, and passport strategies.)
```
npm i express@latest mongoose@latest body-parser@latest bcrypt@latest jsonwebtoken@latest passport@latest passport-jwt@latest passport-local@latest 
```

Add linting capabilities 
```
npm i @eslint/js@latest eslint@latest
```
